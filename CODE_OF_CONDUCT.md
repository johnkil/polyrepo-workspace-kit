# Code of Conduct

Polyrepo Workspace Kit aims to be a focused, respectful project for people working on local developer tooling and agent coordination.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being precise about bugs and risks.
- Keep feedback about the work, not the person.
- Make room for different levels of experience.
- Prefer clear evidence, reproduction steps, and tradeoffs over broad claims.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.
- Pressuring maintainers or contributors to handle private security issues in public.

## Enforcement

The project owner is responsible for moderation and enforcement during the v0.x phase.

Enforcement may include:

- clarifying expectations;
- editing or closing discussions;
- limiting participation;
- escalating to platform moderation tools when needed.

## Reporting

For non-sensitive conduct issues, open a GitHub Issue:

https://github.com/johnkil/polyrepo-workspace-kit/issues

Do not report conduct issues through public threads if doing so would expose private information or create further harm.
